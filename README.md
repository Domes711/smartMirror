# smartMirror

Personal project: MagicMirror² on a Raspberry Pi with face recognition
(profile switching) and an HLK-LD2450 radar (presence-based display on/off).

This repo holds design specs, implementation plans, and a backup of the
user-owned code running on the Pi. Full details — repository layout, target
hardware, external dependencies, conventions — live in [CLAUDE.md](./CLAUDE.md).
